💬 Sentences
我要/想去西雅图。Wǒ yào / xiǎng qù Xīyǎtú.I want to go to Seattle. / I'd like to go to Seattle.
很多 (adj)hěn duōmany / a lot of
很少 (adv)hěn shǎoseldom / rarely
很多人不吃肉。Hěn duō rén bù chī ròu.Many people don't eat meat.
很多人想要很多钱。Hěn duō rén xiǎng yào hěn duō qián.Many people want a lot of money.
她有很多钱。Tā yǒu hěn duō qián.She 有 a lot of money.
我吃了很多包子。Wǒ chī le hěn duō bāozi.I ate a lot of steamed buns.
我喝了很多可乐。Wǒ hē le hěn duō kělè.I drank a lot of cola.
我买了很多书。Wǒ mǎi le hěn duō shū.I bought a lot of books.
我很少喝酒/咖啡。Wǒ hěn shǎo hē jiǔ / kāfēi.I rarely drink alcohol/coffee.
我很少吃羊肉。Wǒ hěn shǎo chī yángròu.I seldom eat lamb.
我很少读书。Wǒ hěn shǎo dúshū.I rarely read books.
我吃了两个包子。Wǒ chī le liǎng gè bāozi.I ate two steamed buns.
我买了两个书包。Wǒ mǎi le liǎng gè shūbāo.I bought two backpacks.
