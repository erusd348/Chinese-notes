📚 Grammar
S adv adj
Subject + Adverb + Adjective. Put an adverb like hěn (very) or tài (too) before an adjective to describe the subject.
我的猫很软。Wǒ de māo hěn ruǎn.My cat is very soft.
这个床垫太软了。Zhège chuángdiàn tài ruǎn le.This mattress is too soft.
太 adj 了
tài + adjective + le = "too" or "so" + adjective.
太好了。Tài hǎo le.That's great!
太好吃了。Tài hǎochī le.It's so delicious!
多 adj
duō + adjective = "how + adjective".
你多大了？Nǐ duō dà le?How old are you?
