📝 Grammar
几点 — Asking the Time
几点 (jǐ diǎn) = "what time?" Structure: 现在 + 几点? Answer: (现在) + number + 点 + minutes.
• 现在几点？   Xiànzài jǐ diǎn?   What time is it now?
• 现在八点半。   Xiànzài bā diǎn bàn.   It's 8:30 now.
Time Expressions
点 (diǎn) = o'clock, 刻 (kè) = quarter-hour, 半 (bàn) = half, 分 (fēn) = minute. 差 (chà) = "to" (as in "ten to eight").
• 七点一刻   qī diǎn yī kè   7:15
• 差十分八点   chà shí fēn bā diǎn   ten to eight (7:50)
Nominal Predicate — No 是 for Time/Date/Age
When the predicate is a time, date, age, or number, do NOT use 是. Just state the time directly.
• 今天六月五号。   Jīntiān liù yuè wǔ hào.   Today is June 5th.
• 现在八点。   Xiànzài bā diǎn.   It's 8 o'clock (now).
从...到... — "From... To..."
从 marks the starting point, 到 marks the end point — for both time and place.
• 从八点到十二点   cóng bā diǎn dào shí'èr diǎn   from 8:00 to 12:00
• 从星期一到星期五   cóng xīngqī yī dào xīngqī wǔ   from Monday to Friday
